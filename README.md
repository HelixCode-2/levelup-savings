# LevelUp Savings

A fun gamified budgeting app for teens.

## Features
- Set savings goals
- Complete daily challenges
- Earn points and level up
- Buy items in the shop
- Go Premium using Paystack (payments in GHS)

## How to Use
1. Open `index.html` in any browser
2. Enter your name to start
3. Start saving and playing!

## How to Install as App
1. Host this folder online (GitHub Pages recommended)
2. Open the link on your phone
3. Tap "Install" or "Add to Home Screen"

Made while learning how to build real apps.
